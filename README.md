# Luminary Studio — Static Layout

A multi-section static marketing page for a fictional creative agency, built as practice.

## Live Preview
[View Live](https://mahaabdulmukhtar.github.io/Luminary-Studio/)

---

## About the Project

Luminary Studio is a fictional design agency. This page was built to practice core HTML and CSS concepts including semantic markup, flexbox, CSS grid, positioning, and responsive design.

---

## Concepts Practiced

| Concept | Where Used |
|---|---|
| Semantic HTML | `header`, `nav`, `main`, `section`, `aside`, `footer` |
| Flexbox | Navigation bar, About section, Footer |
| CSS Grid | Services (3-col), Portfolio (2-col), Contact form |
| Position sticky | Navigation stays at top on scroll |
| Position absolute | Hero badge/tag element |
| CSS Variables | Colors defined in `:root` |
| Responsive design | Media queries collapse layout on mobile |
| Accessibility | `aria-label`, `alt` text, `label` linked to inputs |
| Box model | Card padding, section margins, borders |

---

## Sections

- **Header** — Sticky navigation with logo and links
- **Hero** — Full-width intro with background image
- **Services** — 3-column grid with images and cards
- **About** — Flexbox layout with pull quote in `<aside>`
- **Portfolio** — Asymmetric 2-column image grid
- **Contact** — Form with validation and accessibility
- **Footer** — Flexbox with copyright and social links

---

## Tech Stack

- HTML5
- CSS3 (no frameworks)
